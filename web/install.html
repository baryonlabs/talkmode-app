TalkMode 설치 가이드

처음이라면 5분 안에 — Sequoia (15) · Tahoe (26) 사용자 필수

✓ macOS 14 / 15 / 26 모두 동작 ✓ Apple Silicon (M1 / M2 / M3 / M4) ⚠ 첫 실행 시 시스템 설정에서 한 번 풀어야 함

💡 왜 이 가이드가 필요한가?

TalkMode 는 개인 개발자 인증서 없이 ad-hoc 서명되어 배포됩니다. macOS Sequoia (15) 부터는 이런 앱의 첫 실행이 자동으로 차단되는데, 예전처럼 우클릭 → 열기로는 풀리지 않습니다. 시스템 설정에서 명시적으로 한 번 허용해줘야 합니다. 한 번만 풀면 그 다음부턴 정상 실행됩니다.

1

Homebrew 설치 — 처음이라면 한 번만

Homebrew 는 macOS 용 패키지 관리자입니다. 이미 있으면 2단계로 건너뛰세요.

1-1. 터미널 열기

Spotlight (⌘ + Space) 에서 터미널 검색 → Enter

1-2. 이미 설치돼 있는지 확인

터미널에 다음을 입력:

terminal — bash
$ brew --version
Homebrew 4.x.x
✓ 위와 같이 버전이 나오면 → 2단계로 건너뛰기

대신 brew: command not found 가 나오면 → 아래 한 줄로 설치:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

⏱ 5–15분 소요

설치 중 Mac 로그인 비밀번호와 Command Line Tools 동의 화면이 뜹니다. 안내대로 진행하세요. 끝나면 터미널을 한 번 닫았다 다시 엽니다.

📚 더 자세히: Homebrew 공식 사이트 (한국어) · 블로그 가이드

2

TalkMode 설치

Homebrew 가 준비됐으면 두 줄이면 끝.

terminal — bash
$ brew tap baryonlabs/talkmode
$ brew install --cask talkmode
==> Downloading https://talkmode.baryon.ai/download/TalkMode-0.4.14.zip
==> Installing Cask talkmode
==> Moving App 'TalkMode.app' to '/Applications/TalkMode.app'
🍺 talkmode was successfully installed!
✓ Applications 폴더에 TalkMode.app 가 생겼습니다. Launchpad 또는 Spotlight 에서 TalkMode 검색.
3

첫 실행 — "확인되지 않은 개발자" 경고 풀기 ⚠ 가장 중요

처음 더블클릭하면 아래 경고가 뜹니다. 한 번만 풀어두면 다음부턴 정상 실행됩니다.

3-1. 차단 다이얼로그 → "완료" 클릭

⚠️

"TalkMode"을(를) 열 수 없습니다

TalkMode은(는) 확인되지 않은 개발자가 만든 앱이라 macOS가 안전을 확인할 수 없습니다.

놀라지 마세요 — 정상입니다. "완료" 누르고 다음 단계로.

3-2. 시스템 설정 → 개인정보 보호 및 보안

Spotlight 에서 개인정보 검색 → "개인정보 보호 및 보안" 진입 → 아래로 스크롤

개인정보 보호 및 보안
보안
앱 다운로드 허용: App Store 및 확인된 개발자
"TalkMode"이(가) 차단되었습니다 — 확인되지 않은 개발자가 배포한 앱입니다.
잠시 전 차단됨

👀 여기 "그래도 열기" 버튼을 누르세요

버튼 텍스트는 macOS 버전에 따라 "그래도 열기", "무시하고 열기", "Open Anyway" 중 하나로 나옵니다. 차단된 항목이 보이지 않으면, 방금 TalkMode 더블클릭한 게 맞는지 다시 확인.

3-3. 최종 확인 → "열기"

⚠️

위험이 있어도 이 앱을 실행하시겠습니까?

"TalkMode"은(는) 확인되지 않은 개발자가 만든 앱입니다. 신뢰할 수 있는 출처에서 다운로드한 경우에만 실행하세요.

✓ "열기" 클릭 → TalkMode 가 정상 실행됩니다. 이 단계는 한 번만 거치면 끝 — 다음부턴 그냥 더블클릭하면 바로 열립니다.
4

실행 확인 — 첫 30회는 설정 없이 바로 동작

TalkMode 0.4.14 부터는 기본 LLM 백엔드(Baryon AI 호스팅)가 내장되어 있어 키 설정 없이 30회까지 무료로 사용할 수 있습니다.

  • 마이크 권한 허용 → 말 걸기 시작
  • 30회 후에는 설정 → LLM 제공자 에서 Claude CLI · Codex CLI · OpenAI 키 중 하나로 전환
  • 회의록 / 카메라 시선 추적 등의 권한은 해당 기능을 처음 사용할 때만 요청됨

자주 묻는 질문

brew: command not found 가 계속 나와요

Homebrew 설치가 끝난 뒤에도 터미널을 한 번 닫았다 다시 열어야 PATH 가 적용됩니다. 그래도 안 되면 Homebrew 가 안내한 두 줄의 eval "$(/opt/homebrew/bin/brew shellenv)" 명령을 그대로 복사해 실행하세요.

우클릭 → 열기로 차단을 풀려고 했는데 안 됩니다

macOS Sequoia (15) 부터는 우클릭 → 열기 방법이 더 이상 동작하지 않습니다. 위 3단계의 "시스템 설정 → 개인정보 보호 및 보안" 경로가 유일한 방법입니다.

macOS Ventura (13) 이하에서도 동작하나요?

아니요 — TalkMode 는 macOS Sonoma (14.0) 이상 만 지원합니다. SpeechRecognizer 와 Vision 의 일부 API 가 Sonoma 부터 stable 입니다.

Intel Mac 에서 동작하나요?

현재 0.4.14 는 Apple Silicon (M1 이상) 만 지원합니다. Intel 지원은 빌드 옵션 한 줄과 Cask 메타만 풀면 가능한데, 우선순위는 사용자 분포에 따라 결정 예정. 상세 분석 보고서.

차단 화면에 "그래도 열기" 버튼이 안 보입니다

방금 TalkMode 를 더블클릭해서 차단이 일어났는지 다시 확인하세요. 차단이 일어난 직후에만 시스템 설정의 보안 섹션에 해당 항목이 나타납니다. 다이얼로그를 닫고 일정 시간이 지나면 사라질 수 있어요 — 그러면 다시 TalkMode 더블클릭 → 차단 → 시스템 설정으로 즉시 이동.

이 가이드를 따라도 안 풀려요

경로: GitHub Issues 에 macOS 버전 + 어느 단계에서 막혔는지 + 스크린샷 함께 보고해주세요. 일반적으로 30분 안에 회신드립니다.

마이크 권한을 허용하고 말을 걸어보세요. 한국어로 "안녕" 한 마디면 1초 안에 반응이 옵니다. 회의록 / 브레인스토밍 / 심리상담 모드는 상단 탭에서 전환.
